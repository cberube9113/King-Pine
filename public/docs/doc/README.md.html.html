<div class="pilwrap" id="how-to-run">
  <h3>
    <a href="#how-to-run" class="pilcrow">&#182;</a>
    How to Run
  </h3>
</div>


<p>Our application can be run by running 'node app.js' in the root directory of the program.
Note you must have the required modules which are not included in git:
+ connect-flash
+ ejs
+ express
+ socket.io
+ sequelize</p>

<p>Run 'npm install' in the root directory to install the above modules.</p>

<div class="pilwrap" id="app.js">
  <h4>
    <a href="#app.js" class="pilcrow">&#182;</a>
    app.js
  </h4>
</div>


<ul>
<li>Updated route information to match updates to routes and views.</li>
<li>Added POST for /new-user action, called when a user registers for Chirper.</li>
<li>Added POST for /new-chirp action, called when a logged in user makes a Chirp.</li>
<li>Added POST for /search action, which is called when a user enters a search query.</li>
<li>Added GET for /:user path, which is used in search function.</li>
<li>Added GET for /follow/:user path, which is used to follow/unfollow a user.</li>
<li>Added GET for /spec, which redirects to the functional spec in docs and displays/downloads the PDF of it.</li>
</ul>
